Projects and Publications

My research journey spans NLP, machine learning, and privacy, tackling challenges from video summarization to differential privacy relaxation. I am developing a multi-source academic summarizer leveraging NLP to analyze videos and text, generating concise summaries and discussion prompts. My work on relaxing differential privacy explored improved privacy guarantees while maintaining model performance, and I developed a novel NLP2SQL approach for efficient database interaction. Along with my academic focus, I explored cancer grading via deep learning and network anomaly detection with optimized ANNs, both published in Springer journals.

Publications

Prostate Cancer Grading using Multistage Deep Neural Networks

2023

MIND 2021 • Springer publication

  • Pioneered a high-accuracy (92.38%) deep learning system for automatic prostate cancer grading, surpassing existing methods and enhancing diagnostic efficiency.
  • Championed model explainability through GradCAM visualization, fostering trust and transparency in clinical decision-making.

Network Anomaly Detection using Artificial Neural Networks Optimised with PSO-DE Hybrid

2019

SSCC 2018 • Springer publication

  • Pioneered the PSO-DE hybrid approach for ANN optimization, demonstrating its effectiveness in complex network security tasks.
  • Achieved a 95% F1-score through the novel approach, significantly outperforming existing methods.

Projects

Academic Video and Text summarizer

2023

  • Spearheading the development of a multi-source academic summarizer using NLP to analyze lecture videos & text, generating concise summaries & thought-provoking discussion prompts.
  • Leveraging abstraction-based generative AI to surpass baseline performance in context building and summary accuracy.

Relaxation of Differential Privacy in Machine Learning

2020

  • Analyzed existing Differential Privacy methods to identify limitations and potential for relaxation.
  • A comprehensive literature survey was conducted, and convexity properties of relaxed privacy guarantees were investigated.
  • Successfully applied Renyi DP to machine learning algorithms, balancing individual privacy with model performance.

NLP2SQL using Semi-Supervised Learning and MAPO Policy Gradient

2019

  • Devised a novel approach using semi-supervised learning and MAPO gradient within an LSTM architecture, enhancing model accuracy and efficiency.
  • Efficiently translated natural language (English) queries into SQL for direct database interaction, eliminating manual translation efforts.

packcap: The Network Monitoring Tool

2018

  • Built and deployed a command-line network packet capture tool inspired by Wireshark, enabling efficient network traffic analysis.
  • Utilized the tool to diagnose network anomalies, facilitating problem identification and resolution.
